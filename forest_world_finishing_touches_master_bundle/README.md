## What "V6 Cosmos World Template System" means

V6 is the step where the forest stops being only one handcrafted world and becomes the **reference template**
for all future worlds in the game.

The forest world teaches the reusable pattern:
- environment layer
- elder layer
- attunement layer
- offering layer
- companion layer
- weather layer
- planetary harmonics layer
- resonance memory layer
- portal layer

Then each future destination swaps in its own content:
- forest -> stag, fairies, gnomes, dryads, spring
- tropical -> serpent/jaguar/bird elders, vines, storms, fruiting canopy
- desert/source river -> source wells, solar weather, memory reeds, water shrines
- moon -> magnetic fields, low-gravity flora, eclipse trials, aurora gates

So V6 is not "leave the forest behind."
It means: **finish the forest well enough that it becomes the first cell of the whole cosmos.**
