# V6 Cosmos World Template System

V6 is the next architecture step after the forest is polished.

## Goal
Turn the forest slice into a reusable world-template system.

## Shared structure
Every world uses:
- WorldRoot
- EnvironmentLayer
- ElderLayer
- CompanionLayer
- AttunementLayer
- OfferingLayer
- WeatherLayer
- HarmonicsLayer
- MemoryLayer
- PortalLayer

## Why it matters
Without V6, every planet or moon must be built from scratch.
With V6, the forest becomes the base grammar for all worlds.

## Example expansions
- Forest world: healing / stag / spring / fairies
- Tropical world: lush abundance / serpent elder / storm fruit / river spirits
- Source river world: source wells / light fruit / memory water / guardian shrines
- Moon world: magnetic storms / thin atmosphere / eclipse trials / aurora portals

## Relationship to current work
The current forest finishing touches are valuable because they define the first complete template:
elder + offerings + weather + harmonics + memory + companions + portal outcomes.
